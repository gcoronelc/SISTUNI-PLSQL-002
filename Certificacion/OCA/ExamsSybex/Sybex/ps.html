SYBEX PRODUCT SUPPORT

Sybex has tried to answer any problems that you may experience using the software on this CD. If you do experience problems running this CD and the Readme file doesn't have the information you need, please follow these steps:

  1. Visit the Sybex website at this address: www.sybex.com.
  2. In the search entry box, type the last five digits of the book's ISBN (printed on book cover and CD label) and click GO.
  3. The search engine should retrieve a hyperlink for the book title. Click this link.
  4. If we have made any updates to the Readme file or to the book, they will be available via the web page that the link takes you to. If there is an Update or Errata button on the page, click it.
