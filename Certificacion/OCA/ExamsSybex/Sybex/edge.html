The Sybex Test Engine is a customized exam preparatory program that allows you to deepen your knowledge of the material required for the OCA exam. Test yourself on all of the chapter review questions from the book, gauge your knowledge with the assessment test, or choose from two bonus exams found on only this CD.

System Requirement: A web browser with the lastest Macromedia Flash Player plug-in is needed to run the OCA Sybex Test Engine. 1024x768 or greater display resolution and a high color (16-bit or greater) video card are also required.

To run the OCA Sybex Test Engine, enter the TestEngine folder at the root of this CD and double-click the file named sybex_te.html.
