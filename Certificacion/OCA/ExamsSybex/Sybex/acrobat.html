Adobe Acrobat Reader 7.0 allows you to view portable document format (.pdf) files created in Adobe Acrobat. Included is the Reader for Windows operating systems.

System Requirement: Windows 2000 Professional or Windows XP; for earlier versions of Acrobat, please visit www.adobe.com.

To install Adobe Acrobat Reader, open the Adobe folder, then double-click setup.exe. You will be guided through the installation process.

Note about PDF files: Some users may encounter image blurring in certain files. If so, use the zoom tool to enlarge the view of the blurred area. PDF components of third-party software may be supported by the specific owner(s) of that material but they are not supported by SYBEX.
